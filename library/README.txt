Code Authors: Tom Archer and Chris Wildsmith

Contained within are the Client file needed to run the Python Puzzles RI Masterclass. The files are as follows:

* Client.py - This is the library for the students Pi's in order to allow them to connect to the server as well as send and receive data.
* Lists.py - This is the library that the students use during session 1. They use this to generate a list for their sort algorithm and this also includes a swap function that allows them to pass in a list and the reference of the 2 elements that are to be swapped and this will swap the elements for them.
